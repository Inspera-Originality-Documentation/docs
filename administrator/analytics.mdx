## Introduction

The Analytics section provides administrators with system-wide data and insights into key system events.&#x20;

### Overview

Overview provides administrators with insights into key events and units in the system such as:

* Submission Count

* Assignment Count

* Subject Count

Submissions are also categorized into low, medium, and high similarity.&#x20;
